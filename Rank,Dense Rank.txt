2. Assign sequence order for the student for the same subject based on their marks.
SELECT NAME, SUBJECT, MARKS, RANK() OVER (ORDER BY MARKS DESC) “RANKOFMARKS” FROM STUDENT;

3. Assign sequential order for the student for the same subject based on their marks in descending order.
SELECT NAME, SUBJECT, MARKS, RANK() OVER(PARTITION BY SUBJECT OR BY MARKS DESC)” RANK_OF_SUBJECT_MARKS” FROM STUDENT;

4. Assign sequential order using dense rank function.
SELECT NAME, SUBJECT,MARKS,DENSE_RANK() OVER(ORDER BY MARKS DESC)”DENSE_RANK_OF_MARKS” FROM STUDENT;
